There is no entry for \texttt{2015-06-08} because a standard reduce-side join acts as an inner join.
It requires the date to exist in both datasets simultaneously (\texttt{dropoff\_date} from the taxi dataset and \texttt{businessday} from the fuel dataset) to calculate a result.
If either of the two datasets lacks records for June 8, 2015 (e.g., no taxi trips were recorded or no fuel price entry was logged for that specific date), the join condition fails, and the day is completely omitted from the final output.

\exercise{Hive}
\subexercise{}

The following HiveQL query was used to determine the five most frequently drawn individual lottery numbers.

\begin{verbatim}
SELECT CAST(num AS int) AS number, COUNT(*) AS occurrences
FROM (
    -- Get the space-separated numbers
    SELECT explode(split(lotterynumbers, ' ')) AS num FROM lottery
    UNION ALL
    -- Combine them with the extra number column as string
    SELECT CAST(meganumber AS STRING) AS num FROM lottery 
) t
GROUP BY CAST(num AS INT) --cast to int to avoid "02" and "2" differences
ORDER BY occurrences DESC
LIMIT 5;
\end{verbatim}

The query splits the \texttt{lotterynumbers} column into individual numbers, creates one row per number using \texttt{explode}, counts the occurrences of each number, sorts the result in descending order and returns the top five numbers.

\subexercise{}

To estimate the theoretical gross margin of taxi operations, the aggregated taxi data was joined with the fuel dataset. Following the assumption from Task 2.3, a fuel consumption of 21.3 miles per gallon was used and the Buffalo fuel price was applied.

The following HiveQL query was used:

\begin{verbatim}
SELECT
    t.dropoff_date,
    ((t.total_earnings -
    ((f.buffalo_price * t.total_distance) / 21.3)))
    AS gross_margin_buffalo
FROM (
    SELECT
        dropoff_date,
        SUM(distance) AS total_distance,
        SUM(earnings) AS total_earnings
    FROM taxi
    GROUP BY dropoff_date
) t
JOIN fuel f
ON t.dropoff_date = f.businessday
ORDER BY t.dropoff_date;
\end{verbatim}

The query first aggregates the taxi data by day and then joins it with the fuel dataset using the corresponding date. The theoretical gross margin is calculated as the total earnings minus the estimated fuel costs based on the Buffalo fuel price and a fuel efficiency of 21.3 miles per gallon.
