Setup instructions for AIM excercise on Windows systems

1. Download and install x2go-client from
   https://wiki.x2go.org/doku.php/download:start

2. Extract ZIP-file "aimpm41.zip" to desktop folder
   (Creating folder %USERPROFILE\Desktop\aimpm41)

3. Import session definition
   Enter %USERPROFILE\Desktop\aimpm41 and double-click file setup.cmd

4. Start x2go-client and look for session "aimpm41"
   - Open the session definition for "aimpm41"
   - Check the path to the key-file
   - Optionally increase size of the remote desktop

5. Start session
   - Single-click the session icon
   - Enter password: aim23pm
   - Click OK

6. To finish the session
   - Click on Applications. Then on Log Out.
